## 📌 Overview
This project focuses on analyzing cinema performance data from the Lombardy region using Microsoft Excel. The dataset includes ticket sales and revenue (in USD) across multiple provinces.

The analysis demonstrates core Excel and data analytics skills such as sorting, aggregation, and statistical evaluation.

---

## 📂 Dataset
The dataset contains:
- 🏙️ Province Names  
- 🎟️ Tickets Sold  
- 💰 Total Receipts (USD)  

---

## 🎯 Project Objectives

✔️ Sort provinces by **total takings (descending order)**  
✔️ Calculate **total tickets sold & total revenue**  
✔️ Perform **statistical analysis** (Min, Max, Average)  
✔️ Count total **number of provinces**  
✔️ Compute **average ticket price**

## 📊 Key Analysis Performed

### 🔽 Data Sorting
- Ranked provinces based on highest revenue generation

### ➕ Aggregations
- Calculated:
  - Total Tickets Sold  
  - Total Revenue  

### 📉 Statistical Metrics
- Minimum Value  
- Maximum Value  
- Average Value  

### 🧮 Average Ticket Price
- Formula used:  
  **Average Ticket Price = Total Takings / Total Tickets Sold**

### 📈 Key Insights
- Identified top-performing provinces by revenue  
- Highlighted low-performing regions  
- Observed ticket sales distribution trends  
- Derived average ticket pricing behavior
